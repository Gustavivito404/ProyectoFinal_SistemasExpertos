# Sistema Experto Multiagente para Venta y Mantenimiento de Equipos Contra Incendio

Este proyecto tiene como objetivo desarrollar un sistema experto moderno basado en agentes inteligentes, orientado a la atención de clientes, generación de pedidos, gestión de servicios de mantenimiento y explicación de inferencias dentro del área de equipos contra incendio.

El sistema busca simular el funcionamiento de una plataforma inteligente capaz de interpretar solicitudes de clientes relacionadas con productos como motores, bombas contra incendio, refacciones, tableros eléctricos o servicios de mantenimiento. A partir del mensaje del usuario, el sistema analiza la intención, extrae información relevante, consulta una base de datos local y aplica reglas de inferencia para generar una respuesta, pedido o recomendación técnica.

La propuesta integra conceptos de ingeniería del conocimiento, sistemas expertos, motores de inferencia, bases de datos, agentes inteligentes e inteligencia artificial moderna. El sistema se ejecutará de manera local en computadora personal y contará con una arquitectura modular compuesta por tres agentes principales: un agente de atención al cliente, un agente generador de pedidos o servicios y un agente supervisor encargado de explicar las decisiones tomadas.

Además de automatizar procesos de venta o soporte, el sistema busca ofrecer explicabilidad, mostrando qué reglas fueron activadas y por qué se tomó determinada decisión. Esto permite que el usuario no solo reciba una respuesta automática, sino que también pueda comprender el razonamiento interno del sistema.
